task::Error Class Reference

Detailed Description

A generic exception class carrying a message string.

The Error class is used as the standard exception by many functions throughout Taskolib. It can be used directly or inherited from.

try
{
throw task::Error("An error has occurred");
}
catch (const task::Error& e)
{
std::cerr << e.what() << "\n";
}
