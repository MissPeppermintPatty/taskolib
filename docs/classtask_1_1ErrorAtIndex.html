task::ErrorAtIndex Class Reference

Detailed Description

An exception class storing the index of the step in which an error occurred, in addition to the error message.

try
{
throw task::ErrorAtIndex("An error has occurred", 42);
}
catch (const task::ErrorAtIndex& e)
{
std::cerr << e.what() << " in step " << e.get_index() << "\n";
}
